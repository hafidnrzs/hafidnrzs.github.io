Artikel

Dasar Dasar Komputasi adalah salah satu mata kuliah pada Semester 1 Prodi Teknik Informatika, Fakultas Ilmu Komputer, Universitas Dian Nuswantoro Semarang. Mata Kuliah ini diajar oleh Edi Faisal S.Kom, M.Kom setiap hari Selasa pukul 08.40 - 10.20 di lab D.2.E.

Dasar Dasar Komputasi merupakan mata kuliah yang bertujuan untuk mengenalankan mahasiswa baru dengan teknologi. Mulai dari pengenalan bagian-bagian komputer, cara merakitnya, penggunaan Office, seperti Word, Excel, Power Point, dan Access, serta pengenalan pemrograman web seperti HTML.

Daftar Materi Dasar Dasar Komputasi

  1. Merakit PC Secara Virtual (Cisco)
  2. Penulisan Ilmiah dengan Word
  3. Mailing List dengan Word
  4. Membuat Tabel dengan Excel
  5. Menerapkan Formula dan Chart pada Excel
  6. Membuat Template Power Point
  7. Membuat Database di Access
  8. Fun Code dengan code.org
  9. Pengenalan HTML
  10. Web Design HTML
